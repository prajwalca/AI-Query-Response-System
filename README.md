# AI Query Response System

## Overview

An AI-powered query response system built using Google Forms, Google Sheets, Make.com, and Google Gemini API.

The system accepts user questions through a Google Form, processes them using Gemini AI, and automatically stores the generated responses in a dedicated Google Sheet.

## Workflow

Google Form
→ Google Sheets
→ Make.com Automation
→ Gemini API
→ AI Response Sheet

## Features

* Automated query processing
* AI-generated responses
* Dynamic response generation using Gemini
* Google Workspace integration
* No-code workflow automation using Make.com

## Technologies Used

* Google Forms
* Google Sheets
* Make.com
* Gemini API
* HTTP Requests
* JSON
